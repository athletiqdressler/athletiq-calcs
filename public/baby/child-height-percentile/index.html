Child Height Percentile

Estimate your child's height percentile using reference data from a growth chart (mean and standard deviation).

Use a pediatric growth chart to find the mean and standard deviation, then enter them here.

How it works

This tool assumes height follows a normal distribution. It uses the formula z = (height − mean) / SD and converts z to a percentile.

How to interpret

Percentiles are based only on the reference values you provide. This is an educational helper, not medical advice. For any concern about growth, talk to a pediatrician.

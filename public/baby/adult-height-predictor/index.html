How it works

The mid-parental height formula is a common rule of thumb: for boys, (father + mother + 13) / 2; for girls, (father + mother − 13) / 2. Result is approximate.

How to interpret

Real adult height depends on many factors (nutrition, health, genetics). This tool is for educational use only and does not replace medical evaluation.

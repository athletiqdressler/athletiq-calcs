Child Weight Percentile

Estimate your child's weight percentile using reference mean and standard deviation from a growth chart.

How it works

The calculator applies the normal distribution using the z-score formula z = (weight − mean) / SD and converts z into a percentile.

How to interpret

Percentiles only reflect the reference data you provide. This is not a diagnostic tool. Always discuss growth and weight with a healthcare professional.

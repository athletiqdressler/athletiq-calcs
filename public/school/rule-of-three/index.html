Rule of Three Calculator

Solve proportional problems of the form A is to B as C is to X.

Set up A : B = C : X and the calculator finds X.

How it works

For direct proportion, A : B = C : X. The solution is X = (B × C) / A.

How to interpret

Use it for recipes, scaling quantities, simple financial proportions and school exercises.
